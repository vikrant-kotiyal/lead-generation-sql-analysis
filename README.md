# Lead Generation & Demographic Analysis using SQL

## Project Overview
This project analyzes lead data to understand:
- Lead source performance
- City-wise lead distribution
- Demographic trends

## Dataset
leads_basic_details.csv  
Total records: 360+

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Key SQL Concepts
- Aggregations
- GROUP BY, HAVING
- Subqueries
- CTEs
- Window Functions

## Sample Business Questions
- Which city generates the most leads?
- Which lead source performs best?
- What is the average age by gender?
